This project is a comprehensive microservice-based application designed to streamline the management of financial filings and reporting. It includes services for user management, filing processing, and generating financial reports, providing a robust framework for regulatory and compliance needs.

Key Features

  • Manage user accounts and roles through the User Service.
  • Create, update, and delete filings using the Filing Service.
  • Generate detailed financial reports via the Reporting Service.
  • Seamless integration with a scalable microservices architecture.
