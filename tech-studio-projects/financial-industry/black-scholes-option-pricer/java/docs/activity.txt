5. Activity Diagram

Purpose: To show the workflow or business process.

    Activities:
        User opens Input Form.
        User enters data.
        Frontend validates data.
        Frontend submits data to Backend.
        Backend processes data.
        Backend updates Database.
        Backend sends response to Frontend.
        Frontend updates User with results.
